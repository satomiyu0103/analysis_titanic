最終更新: 2026-06-19

## このリポジトリについて

**Titanic 生存予測のデータ分析プロジェクト**

Kaggle Titanic 形式のデータを用い、EDA・前処理・モデル構築・提出ファイル生成までを行う。
再利用可能なコードは `src/analysis_titanic/`、探索は `notebooks/` で行う。

---

## スコープ

| 項目 | 内容 |
|---|---|
| プロジェクト名 | analysis-titanic |
| 目的 | 乗客の生存 (`survived`) を予測する |
| 技術スタック | Python 3.12+ / uv / pandas / scikit-learn / Jupyter |
| 非スコープ | 本番 API デプロイ・大規模 MLOps |
