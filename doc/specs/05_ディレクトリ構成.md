├── AGENTS.md
├── README.md
├── pyproject.toml
├── uv.lock
├── config/
│   └── .env.example
├── scripts/
│   ├── setup_env.ps1
│   └── setup_env.sh

│   ├── conftest.py

│   └── integration/
├── notebooks/              # EDA・試行錯誤

    ├── ai_guidelines/
    ├── reference/
    ├── adr/
    └── records/

## 主要パスの責務

| パス | 責務 |
|---|---|
| `src/analysis_titanic/` | データ読込・前処理・学習・推論 |
| `notebooks/` | 探索的分析（確定コードは src へ） |
| `data/` | 入力 CSV |
| `models/` | joblib 等のモデル保存 |
| `out/` | submission.csv 等 |
| `tests/` | unit / integration |
| `doc/specs/` | 要件・設計・CHANGELOG |
